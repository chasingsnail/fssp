Ajax请求场景
示例代码 
awsui.ajax.request({
    url : "./jd",
    method : "POST",
    data : {p1:"v1"},
    ok : function(r) {
        alert("server ok!");
    },
    err:function(r){
        alert("server exception!");
    }
});
 注意
1，awsui.ajax.request对应服务端com.actionsoft.bpms.commons.mvc.view.ResponseObject结果处理，能直接处理服务端异常.
2，ResponseObject结果为error、warning，info时自动消息提示。
3，该方法兼容jquery ajax的所有参数和用法。
4，awsui.ajax的url为./jd,这是因为AWSUI根据请求文件的类型不同，而制定的规则，具体URL地址说明如下：
提交场景
url 说明
./w (Web)-请求结果一定是普通Web页面数据时
./jd (Json Data)-请求结果一定是 ResponseObject Json
./xd (Xml Data)-请求结果一定是 ResponseObject XML
./df (Download File)-文件下载
./uf (Upload File)-文件上传
语法差异场景
awsui.ajax用法 jQuery用法
awsui.ajax.request $.ajax
awsui.ajax.load(jq,arguments) $.fn.load(arguments)
awsui.ajax.post $.post
awsui.ajax.get $.get
