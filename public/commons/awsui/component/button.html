按钮

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

  • 响应用户点击行为时。
  • 提供按钮组合，带图标的按钮，还可以进行大小选择。
CSS

通过css类来产生不同的按钮样式，推荐顺序为：theme(主题) -> shape（形状） -> size（大小） -> loading（加载） -> disabled（禁用）

按钮的css类说明如下：

class名称 说明
awsui-btn 用于button元素上的原始样式，默认按钮
awsui-btn-green 用于button元素上的样式，绿色按钮
awsui-btn-blue 用于button元素上的样式，蓝色按钮
awsui-btn-danger 用于button元素上的样式，警告按钮
awsui-btn-warm 用于button元素上的样式，暖色按钮
awsui-btn-disabled 用于button元素上的样式，禁用按钮,不可用状态
awsui-btn-circle 圆形按钮
awsui-btn-icon 带图标的按钮
awsui-btn-lg 控件大小。注：按钮高度大小限制为 large
awsui-btn-sm 控件大小。注：按钮高度大小限制为 small
awsui-loading 按钮处于加载状态
awsui-btn-group 设置按钮组合
按钮主题

按钮有六种主题，从而改变按钮风格。

HTML 代码

<button type="button" class="awsui-btn">默认按钮</button>
<button type="button" class="awsui-btn awsui-btn-green">绿色按钮</button>
<button type="button" class="awsui-btn awsui-btn-blue">蓝色按钮</button>
<button type="button" class="awsui-btn awsui-btn-danger">警告按钮</button>
<button type="button" class="awsui-btn awsui-btn-warm">暖色按钮</button>
<button type="button" class="awsui-btn awsui-btn-disabled">禁用按钮</button>

按钮尺寸

按钮有大、中、小三种尺寸（大awsui-btn-lg、默认、小awsui-btn-sm），高度分别为 32px、28px 和 22px。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue awsui-btn-lg">large size</button>
<button type="button" class="awsui-btn awsui-btn-blue">default size</button>
<button type="button" class="awsui-btn awsui-btn-blue awsui-btn-sm">small size</button>
不可用状态

添加类awsui-btn-disabled可让按钮处于不可用状态，同时按钮样式也会改变。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-disabled">disabled</button>

按钮组合

可以将多个 Button 放入 Button.Group 的容器中并添加类awsui-btn-group。

通过设置 size 为 large small 分别把按钮组合设为大、小尺寸。若不设置 size，则尺寸为中。

	<button type="button" class="awsui-btn">删除</button>
