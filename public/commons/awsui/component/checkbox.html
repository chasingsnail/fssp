Checkbox复选框

在一组可选项中进行多项选择时。

  • 单独使用可以表示两种状态之间的切换，和 switch 类似。区别在于切换 switch 会直接触发状态改变，而 checkbox 一般用于状态标记，需要和提交操作配合。
API
  • $(el).check(config)

config 参数如下：

参数 说明 默认值
checkboxClass 改变checkbox的风格，使用类icheckbox_minimal-(grey/blue/green/yellow/aero)等。 icheckbox_minimal-grey
labelHover 移入文字时是否改变radio的样式 true
labelHoverClass 移入文字时radio的样式 hover

下面这些方法可以用来通过编程方式改变input状态，方法如下：

参数 说明 用例
check 指定当前为选中 $(el).check("check")
uncheck 移除 checked 状态 $(el).check("uncheck")
disable 设置不可用 $(el).check("disable")
enable 设置可用 $(el).check("enable")
toggle 切换状态 $(el).check("toggle")
update 同步状态 $(el).check("update")
destroy 移除样式 $(el).check("destroy")

事件如下：

事件名 说明 用例
ifClicked 被点击时触发 $(el).on("ifClicked",function(){})
ifChanged 选中或状态改变时触发 $(el).on("ifChanged",function(){})
ifChecked 选中触发 $(el).on("ifChecked",function(){})
ifUnchecked 取消选中触发 $(el).on("ifUnchecked",function(){})
ifToggled 状态变化 $(el).on("ifToggled",function(){})
ifDisabled 设置不可用时 $(el).on("ifDisabled",function(){})
ifEnabled 设置可用时 $(el).on("ifEnabled",function(){})

受控的 Checkbox

受控的 Checkbox。

HTML 代码

<input class="awsui-checkbox" checked id="checkboxOpt" value="123456" type="checkbox">
<label class="awsui-checkbox-label" id="checkboxLabel" for="checkboxOpt">checked</label>
<p>
	<button type="button" id="btnChecked" class="awsui-btn awsui-btn-blue">取消选中</button>
	<button type="button" id="btnDisabled" class="awsui-btn awsui-btn-danger">禁用</button>
</p>

JavaScript 代码

$("#btnChecked").on("click", function(){
	if ($("#checkboxOpt").prop("checked")) {
		$("#checkboxOpt").check("uncheck");
		$("#checkboxLabel").html("unchecked");
		$("#btnChecked").text("选中");
	} else {
		$("#checkboxOpt").check("check");
		$("#checkboxLabel").html("checked");
		$("#btnChecked").text("取消选中");
	}
});
$("#btnDisabled").on("click", function(){
	if ($("#checkboxOpt").parent().hasClass("disabled")) {
		$("#checkboxOpt").check("enable");
		$("#btnDisabled").text("禁用");
	} else {
		$("#checkboxOpt").check("disable");
		$("#btnDisabled").text("启用");
	}
});

全选

实现全选效果。

HTML 代码

<p>
	<input class="awsui-checkbox" id="checkboxAll" value="" type="checkbox"/>
	<label class="awsui-checkbox-label">Check all</label>
</p>
<p class="checkAll">
	<input class="awsui-checkbox" value="Apple" type="checkbox"/>
	<label class="awsui-checkbox-label">Apple</label>
	<input class="awsui-checkbox" value="Pear" type="checkbox"/>
	<label class="awsui-checkbox-label">Pear</label>
	<input class="awsui-checkbox" value="Orange" type="checkbox"/>
	<label class="awsui-checkbox-label">Orange</label>
</p>

JavaScript 代码

$("#checkboxAll").on("ifClicked", function(event){
	var $checkbox = $(".checkAll").find(".awsui-checkbox");
	if(event.target.checked){
		$checkbox.iCheck('uncheck');
	}else{
		$checkbox.iCheck('check');
	}
});
点击事件

点击事件使用。

HTML 代码

<input class="awsui-checkbox" id="checkboxClick" value="1" type="checkbox"/>
<label class="awsui-checkbox-label" for="checkboxClick">事件</label>

JavaScript 代码

$("#checkboxClick").on("ifClicked",function(){
	if($(this).prop("checked")){
		$.simpleAlert("不选中");
	} else {
		$.simpleAlert("已选中");
	}
});
change事件

change事件。

HTML 代码

<input class="awsui-checkbox" id="checkboxChange1" name="checkboxChange" value="1" type="checkbox"/>
<label class="awsui-checkbox-label" for="checkboxChange1">选项1</label>
<input class="awsui-checkbox" id="checkboxChange2" name="checkboxChange" value="1" type="checkbox"/>
<label class="awsui-checkbox-label" for="checkboxChange2">选项2</label>

JavaScript 代码

$("input[name='checkboxChange']").on("ifChanged",function(){
	if($(this).prop("checked")){
		$.simpleAlert($(this).attr("id") + "被选中");
	} else {
		$.simpleAlert($(this).attr("id") + "被取消选中");
	}
});
