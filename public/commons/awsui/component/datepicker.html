日期选择框

AWSUI的日期组件的功能使用在于当用户需要输入一个日期时，可以点击标准输入框，弹出日期面板进行选择。其主要是将My97DatePicker作为基础组件，在My97DatePicker原来的日期渲染机制上进行了功能扩展，使AWSUI的日期组件功能更加丰富，更加实用。更多文档请参考 API文档

  • 当用户需要输入一个日期，可以点击标准输入框，弹出日期面板进行选择。
API
  • $(el).datepicker(config)

config 参数如下：

参数 说明 类型 默认值
dateFmt 日期格式 string yyyy-MM-dd
readOnly 只读 boolean false
startDate 设置起始日期 string 当前日期
position 设置弹出位置 object {}
doubleCalendar 设置双日历 boolean false
skin 改变皮肤,可选值（default\twoer\flatred\blue\whyGreen） string twoer
minDate 最小日期 string -
maxDate 最大日期 string -
disabledDays 无效天 array null
disabledDates 无效日期 array null
基本

最简单的用法，在浮层中可以选择或者输入日期。



HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>
日期格式

使用 dateFmt 属性，可以自定义日期显示格式。

也可在 awsui-props 属性中直接来设置，可以自定义日期显示格式。




HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="setDatepickerFmt" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>
<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" value="" class="awsui-datepicker awsui-input" awsui-props="{dateFmt:'yyyy/MM/dd',minDate:'2015/09/09'}"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>

JavaScript 代码

$("#setDatepickerFmt").datepicker({dateFmt:"yyyy-MM-dd HH:mm:ss"});
只读效果

设置readOnly属性 true 或 false 可指定日期框是否只读 。



HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerReadonly" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>

JavaScript 代码

$("#datepickerReadonly").datepicker({
	dateFmt:"yyyy-MM-dd HH:mm:ss",
	readOnly:true
});
起始日期

有时在项目中需要选择生日之类的日期,而默认点开始日期都是当前日期,导致年份选择非常麻烦,你可以通过设置startDate属性,轻松解决此类问题。



HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerStartDate" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>

JavaScript 代码

$("#datepickerStartDate").datepicker({
	startDate:'2002-06-15'
});

日期范围限制

通过设置属性minDate(最小日期),maxDate(最大日期)为静态日期值,来限定日期的范围。

系统提供了$dp.$D和$dp.$DV这两个API来辅助你进行日期运算,此外你还可以通过在 #F{} 中填入你自定义的脚本,做任何你想做的日期限制。






HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerMDate" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>
<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择时间" id="datepickerMTime" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>
<div class="awsui-margin-top-10">
    <div class="awsui-datepicker" style="display: inline-block;">
        <span class="awsui-input-wrapper">
            <input type="text" placeholder="请选择日期" id="datepickerStart" value="" class="awsui-datepicker awsui-input"/>
            <span class="awsui-input-suffix">
                <i class="awsui-iconfont">&#59047;</i>
            </span>
        </span>
    </div>
    <div class="awsui-datepicker" style="display: inline-block;">
        <span class="awsui-input-wrapper">
            <input type="text" placeholder="请选择日期" id="datepickerEnd" value="" class="awsui-datepicker awsui-input"/>
            <span class="awsui-input-suffix">
                <i class="awsui-iconfont">&#59047;</i>
            </span>
        </span>
    </div>
</div>

JavaScript 代码

$("#datepickerMDate").datepicker({
	skin:'whyGreen',
    doubleCalendar:true,
	minDate:'2019-01-10',
	maxDate:'2019-01-28'
});
$("#datepickerMTime").datepicker({
	dateFmt:'H:mm:ss',
	minDate:'8:00:00',
	maxDate:'11:30:00'
});
$("#datepickerStart").datepicker({
	dateFmt:'yyyy-MM-dd',
	readOnly:true,
	minDate:'2019-01-20',
	//maxDate:'#F{$dp.$D(\'datepickerEnd\')}'
	maxDate:'#F{$dp.$D(\'datepickerEnd\')||\'2019-05-20\'}'
});
$("#datepickerEnd").datepicker({
	dateFmt:'yyyy-MM-dd',
	readOnly:true,
	minDate:'#F{$dp.$D(\'datepickerStart\')}',
	maxDate:'2019-05-20'
});
无效天

可以使用此功能禁用周日至周六所对应的日期,通过设置属性:disabledDays (0至6 分别代表 周日至周六)。




HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerDisDays1" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>
<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerDisDays2" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>

JavaScript 代码

$("#datepickerDisDays1").datepicker({
	disabledDays:[6]
});
$("#datepickerDisDays2").datepicker({
	disabledDays:[0,6]
});
无效日期

通过设置属性:disabledDates ,使用正则表达式,你可以尽情发挥来实现无效日期。




HTML 代码

<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerDisData1" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>
<div class="awsui-datepicker">
	<span class="awsui-input-wrapper">
		<input type="text" placeholder="请选择日期" id="datepickerDisData2" value="" class="awsui-datepicker awsui-input"/>
		<span class="awsui-input-suffix">
			<i class="awsui-iconfont">&#59047;</i>
		</span>
	</span>
</div>

JavaScript 代码

$("#datepickerDisData1").datepicker({
	disabledDates:['3$']
});
$("#datepickerDisData2").datepicker({
	minDate:'%y-%M-01',
	maxDate:'%y-%M-%ld',
	disabledDates:['0[4-7]$','1[1-5]$','2[58]$']
});
