日期范围

AWSUI的日期组件的功能使用在于当用户需要输入一个日期范围时，可以点击标准输入框，弹出日期面板进行选择。其主要是将My97DatePicker作为基础组件，在My97DatePicker原来的日期渲染机制上进行了功能扩展，使AWSUI的日期组件功能更加丰富，更加实用,更多文档请参考 API文档

API
  • $(el).datepicker(config)

config 参数如下：

参数 说明 类型 默认值
id 唯一标识 string -
name 名称 string -
placeholderr 提示信息 string -
style 样式 string -
title 提示框 string -
callbackfun 选择日期时触发的函数 function -
startTime 选择的开始日期 string -
endTime 选择的结束日期 string 当前日期
showLeft 日期选择浮动层出现的左距离 number -
showTop 日期选择浮动层出现的顶部距离 number -
emptyBtnt 是否显示清空按钮 boolean true
isShowToday 是否显示今天 boolean true
引入文件

务必确保路径正确

<script src="../commons/js/jquery/scripts/ui/datepicker/aws.datepicker-zh-CN.js"></script>
