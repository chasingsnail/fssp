输入框

在输入框中单击鼠标会出现插入点光标，我们可以直接在输入框中输入文字或文本信息。使用场景：

  • 需要用户输入表单域内容时。
  • 提供组合型输入框，带图标（如搜索）的输入框，还可以进行大小选择。
CSS
class名称 说明
awsui-input 用于input元素上的主样式，也是默认样式
awsui-input-lg 控件大小。注：标准表单内的输入框大小限制为 large
awsui-input-sm 控件大小。注：标准表单内的输入框大小限制为 small
awsui-input-group 设置固定组合
awsui-input-group-addon 设置固定组合的前后置标签
awsui-input-wrapper 设置前后缀图标的父级元素
awsui-input-prefix 前缀图标
awsui-input-suffix 后缀图标
Attribute
属性名称 说明
disabled 禁用效果
readonly 只读效果
基本使用

基本使用。

<input type="text" placeholder="Basic usage" class="awsui-input"/>
三种大小

我们为 <Input /> 输入框定义了三种尺寸（大awsui-input-lg、默认、小awsui-input-sm），高度分别为 32px、28px 和 22px。

<Input />宽度默认100%，若有特殊要求要指定宽度，请自行指定。

<div class="awsui-example-input">
	<input type="text" placeholder="large size" class="awsui-input awsui-input-lg">
	<input type="text" placeholder="default size" class="awsui-input">
	<input type="text" placeholder="small size" class="awsui-input awsui-input-sm">
<div>
前置/后置标签

用于配置一些固定组合。



Http:// .com

HTML 代码

//前置标签
<span class="awsui-input-group">
	<span class="awsui-input-group-addon">
		<span class="awsui-iconfont">&#58933;</span>
	</span>
	<input type="text" class="awsui-input" value="localhost"/>
</span>
//后置标签
<span class="awsui-input-group">
	<input type="text" class="awsui-input" value="localhost"/>
	<span class="awsui-input-group-addon">
		<span class="awsui-iconfont">&#58933;</span>
	</span>
</span>
//前后置标签
<span class="awsui-input-group">
	<span class="awsui-input-group-addon">Http://</span>
	<input type="text" class="awsui-input" value="localhost"/>
	<span class="awsui-input-group-addon">.com</span>
</span>
//带select前后置标签
<span class="awsui-input-group">
	<span class="awsui-input-group-addon awsui-select-group awsui-select-group-noborder-right"><select id="demo1"></select></span>
	<input type="text" class="awsui-input" value="localhost"/>
	<span class="awsui-input-group-addon awsui-select-group awsui-select-group-noborder-left"><select id="demo2"></select></span>
</span>

JavaScript 代码

$(document).ready(function() {
	var opt = {
		width: "100%",
		data: [
			{id: "http1", text: "Http://"},
			{id: "http2", text: "Https://"},
		]
	};
	$("#demo1").select2(opt);
	var opt2 = {
		width: "100%",
		data: [
			{id: "com1", text: ".com"},
			{id: "com2", text: ".jp"},
			{id: "com3", text: ".cn"},
			{id: "com4", text: ".org"}
		]
	};
	$("#demo2").select2(opt2);
});



输入框组合

输入框的组合展现。

注意：使用 compact 模式时，不需要通过 Col 来控制宽度。

<div class="awsui-input-group">
	<span class="awsui-col-md-3 awsui-col-sm-3 awsui-col-xs-3">
		<input type="text" value="123456" class="awsui-input"/>
	</span>
	<span class="awsui-col-md-6 awsui-col-sm-6 awsui-col-xs-6">
		<input type="text" value="123456789" class="awsui-input"/>
	</span>
</div>

<span class="awsui-input-group awsui-input-group-compact">
	<input type="text" class="awsui-input awsui-border-no-right" value="1234" style="width: 20%;"/>
	<input type="text" class="awsui-input" value="123456789" style="width: 30%;"/>
</span>

<span class="awsui-input-group awsui-input-group-compact">
	<input type="text" class="awsui-input awsui-border-no-right" value="123" style="width: 20%;"/>
	<input type="text" class="awsui-input" value="123456" style="width: 30%;"/>
	<input type="text" class="awsui-input awsui-border-no-left" value="123456789" style="width: 40%;"/>
</span>
文本域

用于多行输入，使用textarea标签,通过设置属性rows=""来指定行数。

<textarea type="textarea" rows="4" class="awsui-input"></textarea>
禁用效果

禁止和只读效果,添加属性disabled和readonly。

<input type="text" value="禁止效果" disabled class="awsui-input"/>
<input type="text" value="只读效果" readonly class="awsui-input"/>
