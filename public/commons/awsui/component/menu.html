菜单

菜单是一个网站的灵魂，用户依赖导航在各个页面中进行跳转。一般分为顶部导航和侧边导航，顶部导航提供全局性的类目和功能，侧边导航提供多级结构来收纳和排列网站架构。

API
  • $(el).menu(config);

config 参数如下：

参数 说明 类型 默认值
target 菜单目标位置，如果为空，菜单项的起始位置是父容器的左上角 object null
useTargetPosition 是否使用菜单目标位置，若为false，菜单项起始位置是父容器左上角 boolean true
position 菜单位置，选择值为left、center、right string left
top 顶点距父容器的距离，target为空则失效 number 0
left 左侧距父容器距离，target为空则失效 number 0
items 菜单显示列表项，可以是从后台传来的 array []

$(el).menu.items.props

参数 说明 类型 默认值
text 菜单项文本内容 string -
iconCls 列表项图标类,可选值有icon-add、icon-edit、icon-remove、icon-home、icon-menu、icon-save、icon-search、icon-refresh等。 string -
iconFont 列表项引用字体库图标
{
  code:"", //图标编码
  color:"", //设置图标颜色
  fontSize:"14px", //设置图标大小，默认14px
} 		object iconFont:{}
icon 自定义图标 string -
method 列表项事件 function -
disabled 设置菜单项禁用效果 boolean false

JavaScript 代码

$("#btnBasic").click(function(){
	$("#menuBasic").menu({ //初始化菜单
		target:$("#btnBasic")
	})
});
动态菜单

菜单显示列表项，可以是从后台传来的。

    HTML 代码

    <button id="btnItems" type="button" class="awsui-btn awsui-btn-blue">点击触发菜单</button>
	<ul id="menuItems" class="awsui-menu"></ul>

    JavaScript 代码

    $("#btnItems").click(function(){
	$("#menuItems").menu({ //初始化菜单
		target:$("#btnItems"),
		items:[
			{text:"添加", iconCls:"icon-add"},
			{text:"关闭", iconCls:"icon-remove"}
		]
	});
});
    引用字体库图标

    列表项加图标，iconfont字体库图标使用。

			{text:"编辑", iconFont:{code:"&#58956;",color:"#f5bf46",fontSize:"18px"}}
		]
	});
});

        受控操作示例

        受控操作示例。

          HTML 代码

          <button id="btnItemsEvent" type="button" class="awsui-btn awsui-btn-blue">点击触发菜单</button>
	<ul id="menuItemsEvent" class="awsui-menu"></ul>

          JavaScript 代码

          $("#btnItemsEvent").click(function(){
	var getText = function(){
		alert($(this).text());
	};
	$("#menuItemsEvent").menu({ //初始化菜单
		target:$("#btnItemsEvent"),
		items:[
			{text:"添加", iconCls:"icon-add", method:getText},
			{text:"关闭", iconCls:"icon-remove", method:getText}
		]
	});
});
