里程碑

对流程进度的展示和编辑。

API
  • $(el).milestone(config);

config 参数如下：

参数 说明 类型 默认值
width 宽度 number -
height 高度 number -
showMaxNum 显示最大数 number -
defaultColor 设置进度状态色值 object defaultColor:{}
data 数据 object data:[{id:"1",name:"流程1",status:1}]
animateTime 启用动画后的动画时间 string 150ms
readonly 只读效果 boolean false
allowClear 是否清除 boolean false

属性如下:

属性 说明 类型
readonly 只读效果设置 boolean
setWidth 设置宽度 number
基本

最简单的用法。

HTML 代码

<div class="awsui-code-demo">
    <input type="text" id="milestoneBasic" style="display: none">
</div>

JavaScript 代码

var options={
    width:600,
    height:25,
    showMaxNum:5,
    defaultColor:{
        1:"#1CB789",
        2:"#0494D1",
        3:"#e0e0e0",
        4:"#fff",
        5:"#FFFFFF",
        6:"#000"
    },
    data:[
        {id:"1",name:"流程1",status:1},
        {id:"2",name:"流程2",status:1},
        {id:"3",name:"流程3",status:2},
        {id:"4",name:"流程4",status:3},
        {id:"5",name:"流程5",status:3},
        {id:"6",name:"流程6",status:2},
        {id:"7",name:"流程7",status:2},
        {id:"8",name:"流程8",status:6}
    ],
    readonly:false,
    allowClear: false
};
$("#milestoneBasic").milestone(options);
