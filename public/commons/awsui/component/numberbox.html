数字输入框

数字框（numberbox）用于让用户仅能输入数字的值，它可以把输入元素转换为不同类型的输入（比如：数字 number、百分比 percentage、货币 currency，等等）。

  • 当需要获取标准数值时。
API
  • $(el).numberbox(config)

config 参数如下：

参数 说明 类型 默认值
min 最小值 number 0
max 最大值 number 100
uplength 增量或减量 number 1
defaultValue 默认值 number -
symbol 单位 string -
width 宽度 number -
size 输入框大小,当 size 分别为 large 和 small 时，输入框高度为 32px 和 22px，默认高度为 28px string -
disabled 禁用 boolean false
onChange 变化回调 function -
基本

数字输入框。

HTML 代码

<input type="text" id="number1"/>

JavaScript 代码

$("#number1").numberbox();
三种大小

三种大小的数字输入框，当配置的参数size 分别为 large 和 small 时，输入框高度为 32px 和 22px ，默认高度为 28px。

HTML 代码

<input type="text" id="numberLarge"/>
<div class="awsui-number">
	<input type="text" id="numberDefault"/>
</div>
<div class="awsui-number">
	<input type="text" id="numberSmall"/>
</div>

JavaScript 代码

$("#numberLarge").numberbox({
	width:100,
	size:"large"
});
$("#numberDefault").numberbox({
	width:100
});
$("#numberSmall").numberbox({
	width:100,
	size:"small"
});
不可用

设置属性disabled:true，点击按钮切换可用状态。

HTML 代码

<input type="text" id="numberDisabled"/>
<button type="button" class="awsui-btn awsui-btn-blue" id="disabledBtn">toggle disabled</button>

JavaScript 代码

 $("#numberDisabled").numberbox({
	defaultValue:3,
	disabled:true
});
$("#disabledBtn").on("click", function(){
	if($("#numberDisabled").attr("disabled")) {
		$("#numberDisabled").numberbox({
			disabled:false
		});
	} else {
		$("#numberDisabled").numberbox({
			disabled:true
		});
	}
});
最大、最小值设置

最大、最小值设置，设置属性max和min来控制。

设置属性uplength来增量或减量。

HTML 代码

<input type="text" id="numberMvalue"/>

JavaScript 代码

 $("#numberMvalue").numberbox({
	max:8,
	min:3,
	uplength:0.5
});
单位

展示具有具体含义的数据，设置属性symbol来控制。

HTML 代码

<input type="text" id="numberSymbol"/>

JavaScript 代码

 $("#numberSymbol").numberbox({
	symbol:"%"
});
