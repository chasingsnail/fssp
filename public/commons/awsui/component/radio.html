Radio单选框

用于在多个备选项中选中单个状态。

  • Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。
API
  • $(el).check(config)

config 参数如下：

参数 说明 默认值
radioClass 改变radio的风格，使用类iradio_minimal-(grey/blue/green/aero)等。 iradio_minimal-grey
labelHover 移入文字时是否改变radio的样式 true
labelHoverClass 移入文字时radio的样式 hover

下面这些方法可以用来通过编程方式改变input状态，方法如下：

参数 说明 用例
check 指定当前为选中 $(el).check("check")
uncheck 移除 checked 状态 $(el).check("uncheck")
disable 设置不可用 $(el).check("disable")
enable 设置可用 $(el).check("enable")
toggle 切换状态 $(el).check("toggle")
update 同步状态 $(el).check("update")
destroy 移除样式 $(el).check("destroy")

事件如下：

事件名 说明 用例
ifClicked 被点击时触发 $(el).on("ifClicked",function(){})
ifChanged 选中或状态改变时触发 $(el).on("ifChanged",function(){})
ifChecked 选中触发 $(el).on("ifChecked",function(){})
ifUnchecked 取消选中触发 $(el).on("ifUnchecked",function(){})
ifToggled 状态变化 $(el).on("ifToggled",function(){})
ifDisabled 设置不可用时 $(el).on("ifDisabled",function(){})
ifEnabled 设置可用时 $(el).on("ifEnabled",function(){})
基本

最简单的用法，指定当前是否选中。

HTML 代码

<input class="awsui-radio" name="radiox1" id="radio1" value="123456" type="radio" >
<label class="awsui-radio-label" for="radio1">选项1</label>
<input class="awsui-radio" name="radiox1" id="radio2" value="123456" type="radio">
<label class="awsui-radio-label" for="radio2">选项2</label>

JavaScript 代码

$("#radio2").check("check");

启用禁用

启/禁用单选框。

HTML 代码

<input class="awsui-radio" id="radioAble" value="123456" checked type="radio">
<label class="awsui-radio-label" for="radioAble">选项1</label>
<p>
	<button type="button" id="btnEnable" class="awsui-btn awsui-btn-blue">启用</button>
	<button type="button" id="btnDisable" class="awsui-btn awsui-btn-danger">禁用</button>
</p>

JavaScript 代码

$("#btnEnable").on("click", function(){
	$("#radioAble").check("enable");
});
$("#btnDisable").on("click", function(){
	$("#radioAble").check("disable");
});

Radio单选框的风格,配置radioClass属性。

HTML 代码

<input class="awsui-radio" checked id="radioStyle" name="radioStyle" value="1" type="radio">
<label class="awsui-radio-label" for="radioStyle">默认</label>

<label class="awsui-radio-label" for="radioStyle1">蓝色</label>
<input class="awsui-radio" checked id="radioStyle2" name="radioStyle2" value="1" type="radio">
<label class="awsui-radio-label" for="radioStyle2">绿色</label>
<input class="awsui-radio" checked id="radioStyle3" name="radioStyle3" value="1" type="radio">
<label class="awsui-radio-label" for="radioStyle3">红色</label>

JavaScript 代码

$("#radioStyle1").check({
	radioClass : 'iradio_minimal-blue',
	labelHover: false
});
$("#radioStyle2").check({
	radioClass : 'iradio_minimal-green'
});
$("#radioStyle3").check({
	radioClass : 'iradio_minimal-red'
});
