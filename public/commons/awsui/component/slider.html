滑块

当用户需要在数值区间/自定义区间内进行选择时，可为连续或离散值。

引入文件

务必确保路径正确

<script src="../commons/js/jquery/scripts/ui/slider/css/green/green.css"></script>
<script src="../commons/js/jquery/scripts/ui/slider/js/range.js"></script>
<script src="../commons/js/jquery/scripts/ui/slider/js/timer.js"></script>
<script src="../commons/js/jquery/scripts/ui/slider/js/slider.js"></script>
API
  • var slider = new Slider($(parentElement)[0], $(childElement)[0]);

属性如下：

属性 说明 类型 默认值
placeholder 组件提示信息 string -
style 组件样式 string -
title 大部分组件会将该属性渲染成提示框 string -
min 设置表示最低位置的数字 number 0
max 设置表示最高位置的数字 number 100

方法如下：

方法名 说明 用例
setValue 赋值方法 slider.setValue(val)
getValue 获取值 slider.getValue()
setMinimum 设置最小值 slider.setMinimum(minVal)
getMinimum 获取最小值 slider.getMinimum()
setMaximum 设置最大值 slider.setMaximum(maxVal)
getMaximum 获取最大值 slider.getMaximum()
setBlockIncrement 设置最小位移量 slider.setBlockIncrement(val)
onchange 监听滑块change事件 slider.onchange=function(){}
