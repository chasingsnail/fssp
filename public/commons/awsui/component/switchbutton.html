开关

开关选择器，切换开关会直接触发状态改变。

  • 需要表示开关状态/两种状态之间的切换时。
  • 和 checkbox的区别是，切换 switchButton 会直接触发状态改变，而 checkbox 一般用于状态标记，需要和提交操作配合。
API
  • $(el).switchButton(config)

config 参数如下：

参数 说明 类型 默认值
color 打开状态背景颜色 string #64bd63
secondaryColor 关闭状态背景色 string #dfdfdf
jackColor 开关按钮颜色 string #ffffff
onColor 开字体颜色 string #ffffff
offColor 关字体颜色 string #000000
className 开关类名 string Switchery
showtextflag 是否显示开关文字 boolean true
disabled 不可用 boolean false
disabledOpacity 不可用透明度(disabled为true)才有效 number 0.5
speed 开关切换时间(0.1的偶数倍) string 0.1s
ontext 开显示文字 string 开
offtext 关显示文字 string 关
swwidth 宽度 numbert 80
swheight 高度 numbert 28
fontSize 字体大小 numbert 13
基本

最简单的用法。

HTML 代码

<input type="checkbox" id="switchBasic"/>

JavaScript 代码

$("#switchBasic").switchButton();
不可用

switchButton 失效状态。

HTML 代码

<input type="checkbox" id="switchDisabled"/>

JavaScript 代码

$("#switchDisabled").switchButton({
	disabled : true,
	disabledOpacity : 0.5
});
自定义开关

switchButton 自定义开关。

HTML 代码

<input type="checkbox" id="switchCustom"/>

JavaScript 代码

$("#switchCustom").switchButton({
	showtextflag : true,//显示文字
	color : '#5FB878',//开背景色
	secondaryColor : '#f5f5f5',//关背景色
	jackColor : '#FFFFFF',//按钮色
	onColor : '#fff', //开字体颜色
	offColor : '#000',//关字体颜色
	ontext : "打开",//开显示文字
	offtext : "关闭",//关显示文字
	swwidth : 80,//宽度
	swheight : 30,//高度
	fontSize : 13//字体大小
});
获取开关状态

switchButton 获取开关状态。

HTML 代码

<p>
	<input type="checkbox" id="switchStatus"/>
</p>
<p>
	<input type="text" id="statusText"/>
</p>
<button id="getStauts" type="button" class="awsui-btn awsui-btn-blue">
	获取状态
</button>

JavaScript 代码

$("#switchStatus").switchButton({
	change : function() {
		$("#statusText").val($("#switchStatus").prop("checked"));
	}
});
$("#getStauts").on("click", function() {
	alert("checked: " + $("#switchStatus")[0].checked);
});
文字和图标

带有文字和图标。

HTML 代码

<input type="checkbox" id="switchSetNoFont"/>
<input type="checkbox" id="switchSetFont"/>
<input type="checkbox" id="switchSetNum"/>
<input type="checkbox" id="switchSetIcon"/>

JavaScript 代码

$("#switchSetNoFont").switchButton({
	ontext : "",//开显示文字
	offtext : ""
});
$("#switchSetFont").switchButton({
	ontext : "开",//开显示文字
	offtext : "关",
	offColor : '#666',//关字体颜色
	fontSize : 12//字体大小
});
$("#switchSetNum").switchButton({
	ontext : "1",//开显示文字
	offtext : "0",
	offColor : '#666',//关字体颜色
	fontSize : 16//字体大小
});
$("#switchSetIcon").switchButton({
	ontext : "<i class='awsui-iconfont'>&#xe639;</i>",//开显示文字
	offtext : "<i class='awsui-iconfont' style='font-size: 10px;'>&#59134;</i>",
	offColor : '#666'//关字体颜色
});
