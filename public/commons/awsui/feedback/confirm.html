确认框

目标元素的操作需要用户进一步的确认时，在当前页面正中打开一个确认框，询问用户。

API
  • $.confirm(config)

config 参数如下：

参数 说明 类型 默认值
title 确认框标题 string -
content 需要显示的内容 string -
model 是否有遮罩 boolean true
width 弹出层宽度 string -
height 弹出层高度 string -
type 消息类型,值可以为confirm/alert string confirm
secondaryConfirm 是否支持二次确认 boolean false
secondaryConfirmInfo 二次确认信息，secondaryConfirm为true时有效 string -
onConfirm 确认事件 function -
onClose 取消事件 function -
标准confirm

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue"  id="confirm1">标准confirm</button>

JavaScript 代码

$("#confirm1").on("click", function(){
    var options = {
        title : "确认框标题",
        content:"确认框内容",
        onConfirm: function(){
            $.simpleAlert("您选择了确认");
        },
        onClose : function(){
            $.simpleAlert("您选择了取消");
        }
    };
    $.confirm(options);
});

        },
        onClose : function(){
            $.simpleAlert("您选择了取消");

    };
    $.confirm(options);
 });
