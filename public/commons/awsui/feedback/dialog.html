对话框

需要用户处理事务，又不希望跳转页面以致打断工作流程时，可以使用 dialog 在当前页面正中打开一个浮层，承载相应的操作。

  • 当需要一个简洁的确认框询问用户时。
  • 带有交互的通知，给出用户下一步的行动点。
API
  • $(el).dialog(config)

config 参数如下：

参数 说明
id 需组件唯一标识
name 组件名称
style 组件样式
closable 是否可关闭,默认为true
draggable 是否可拖拽,默认为true
model 是否有遮罩层,默认为true
buttons 对话框按钮组，值为[{text:'确定',cls:"blue",handler:function(){}}]
width 对话框宽度，默认80%
height 对话框高度，默认适应内容
fixed 对话框是否相对窗口绝对定位,默认为true
scrollDiv 阻止滚动事件溢出的对象选择器($("#targetDiv")),使用该属性须指定对话框高度
方法说明
方法 说明
close 窗口关闭方法，用法为：$("x").dialog("close")
option,title 设置窗口标题方法，用法为：$("x").dialog("option","title","xx")
onClose 窗口关闭前事件，返回false可阻止关闭
调整

可以通过调整类来修改dialog对话框宽度,如若宽度不合适，调用dialog()方法时设置属性width来调整到适合的宽度。

有以下2种不同的尺寸可选:

class名称 宽度
awsui-size-large 60%
awsui-size-x-large 80%
标准dialog
这是对话框的内容。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="default">点击打开标准对话框</button>
 <div title="这是一个标准的Dialog" class="awsui-size-x-large awsui-hide" id="dialog-normal">
	这是对话框的内容。
 </div>

JavaScript 代码

$("#default").on("click", function(){
    $("#dialog-normal").dialog({
		height:200,
		buttons:[
			{text:'确定',cls:"blue",handler:function(){alert('确定');}},
			{text:'取消',handler:function(){
					//alert('取消');
					this.disabled = true;
				}
			}
		]
	});
});

组合dialog
这是对话框的内容。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="iframe_demo">点击打开iframe</button>
 <div title="这是一个组合的Dialog" class="awsui-size-x-large awsui-hide" id="dialog-iframe">
	这是对话框的内容。
 </div>

JavaScript 代码

$("#iframe_demo").on("click", function(){
    var dlg=FrmDialog.open({
        width:800,
        height:700,
        url:"#",
        data:{}
    });
});

 说明

1.支持一个window中打开多个

2.当有多个iframe时,通过id区分,即 var dlg=FrmDialog.open({width:700,url:url,data:{},id:"MyID"});

3.实例对象的方法： var dlg=FrmDialog.get("MyID")

4.设置实例name属性： dlg.name=FrmDialog.name()

5.设置实例win属性： dlg.win=FrmDialog.win())

6.设置实例id属性： dlg.id=FrmDialog.id("id")

7.打开对话框，可设置dialog组件的各种参数，例如title、onClose、width等

8.url:可选参数，目标地址。如果没有此参数，需自行处理提交

9.data:可选参数，请求参数

10.关闭对话框: FrmDialog.close();可选参数，请求参数
