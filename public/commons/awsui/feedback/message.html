消息提示

  • 可提供成功、警告和错误等反馈信息。
  • 顶部居中显示并自动消失，是一种不打断用户操作的轻量级提示方式。
API
  • $.simpleAlert(config)

config 参数如下：

参数 说明
msg 需要显示的内容。
alertType 提示框的类型，其值可以是info,error,ok,warning,loading。loading类型的提示默认不会关闭，需要开发人员调用$.simpleAlert("close")关闭。
delay 延迟关闭时间(毫秒),为空使用默认值,只对info、ok两种有效,若指定no则消息不关闭
options
mode:是否打开遮罩层。
callback:消息关闭后的回调函数。
forceClose:在警告、错误消息类型下,默认手动关闭,设置true，按照指定delay时间后自动关闭
标准Message

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" value="info" id="info">INFO</button>

JavaScript 代码

$("#info").on("click", function(){
    $.simpleAlert("Hello, 这是一个标准的消息框");
});
遮罩Message

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" value="info" id="info2">
    带遮罩的info消息框
</button>

JavaScript 代码

$("#info2").on("click", function(){
    $.simpleAlert("Hello, 这是一个遮罩的消息框","info",2000,{mode:true});
});
回调Message

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" value="info" id="info3">
    带回调函数的消息框
</button>

JavaScript 代码

$("#info3").on("click", function(){
    $.simpleAlert("Hello, 带回调函数的消息框,时间到触发","info",2000,{callback:myCallback});
});
触发回调函数

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" value="info" id="info4">
    触发回调函数
</button>

JavaScript 代码

$("#info4").on("click", function(){
    $.simpleAlert("Hello, 触发回调函数","info","no",{
        callback:function(){
            console.log('I was closed.');
        }
    });
});

默认不会关闭，需要开发人员调用$.simpleAlert(\"close\")关闭
该提示会在5s后关闭",'loading');
        setTimeout('$.simpleAlert("close")',5000);

    }
