消息页面

使用页面反馈重要信息。通常适用于用户提交某项操作时，返回特定页面提醒，使用message-type="no_content"、message-type="no_search"提供更加友好的引导语，并且根据父级元素的高度来按比例缩小的，在使用代码时父级元素一定要设定高度。适用场景如：

  • 操作被拒绝
  • 无用户数据/信息
  • 没有查询到结果
Type
图标 message-type 场景
API
  • awsuiMessagePage()
注意

 如果是js渲染的html代码，请渲染代码完成后调用 awsuiMessagePage()方法；若html代码直接写在页面中的，不需要调用此方法。 如下： 

var html = $('<div class="awsui-message-page">'+
'<div class="content">'+
	'<span class="icon" message-type="ok"></span>'+
	'<span class="title">已完成</span>'+
	'</div>'+
'</div>');
$(el).append(html);

awsuiMessagePage();
