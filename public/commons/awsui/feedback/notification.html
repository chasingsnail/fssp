通知提醒

通知作为反馈和确认机制。在系统四个角显示通知提醒信息，经常用于以下情况：

  • 较为复杂的通知内容。
  • 带有交互的通知，给出用户下一步的行动点。
  • 系统主动推送。
API
  • $.notification(config)

config 参数如下：

参数 说明 类型 默认值
title 通知提醒标题，必选 string -
description 通知提醒内容，必选 string -
delay 默认 3 秒后自动关闭，配置为 0 则不自动关闭 number 3s
icon 字体库图标 （需要的图标请参考字体库） string -
color 字体库图标颜色 string green
img 自定义图标 （图片路径） string -
fontSize 字体库图标大小 number 30
btn 自定义关闭按钮 string -
position 弹出位置，可选 topLeft topRight bottomLeft bottomRight string topRight
onClose 触发回调函数 function -
基本

最简单的用法，3秒后自动关闭。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="notify_basic"><span>notification</span></button>

JavaScript 代码

$("#notify_basic").on("click", function(){
	$.notification({
		title:'通知标题',
		description:'通知描述。'
	});
});
自动关闭的延时

自定义通知框自动关闭的延时，默认3s，取消自动关闭只要将delay该值设为 0 即可。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="notify_delay"><span>notification</span></button>

JavaScript 代码

$("#notify_delay").on("click", function(){
	$.notification({
		title:'通知标题',
		description:'这是通知的内容。',
		delay:0,
		onClose:function(){
			console.log('I was closed.');
		}
	});
});

HTML 代码

$("#notify_ok").on("click", function(){
    $.notification({
        title:'通知标题',
        description:'这是通知的内容。',
        position: 'topRight',
        icon:"&#60017;" ,
        color:'green'
    });
});

自定义按钮

自定义关闭按钮的样式和文字,设置属性btn即可。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="notify_btn"><span>notification</span></button>

JavaScript 代码

$("#notify_btn").on("click", function(){
	$.notification({
		title:'通知标题',
		description:'这是通知的内容。',
		btn:'Confirm'
	});
});
自定义图标

图标可以被自定义，将图片路径赋给img属性即可。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="notify_icon"><span>notification</span></button>

JavaScript 代码

$("#notify_icon").on("click", function(){
	$.notification({
		title:'通知标题',
		description:'这是通知的内容。',
		img:'../../awsui/img/jq.png'
	});
});
位置

通过设置position属性，可以设置通知从右上角(topRight)、右下角(bottomRight)、左下角(bottomLeft)、左上角(topLeft)弹出。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="notify_position"><span>open the notification box</span></button>

JavaScript 代码

$("#notify_position").on("click", function(){
	$.notification({
		title:'通知标题',
		description:'这是通知的内容。',
		position: "topLeft"
	});
});
