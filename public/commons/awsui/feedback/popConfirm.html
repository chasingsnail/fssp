气泡确认框

目标元素的操作需要用户进一步的确认时，在目标元素附近弹出浮层提示，询问用户。 和 confirm 弹出的全屏居中模态对话框相比，交互形式更轻量。

  • 需要用户进一步的确认时。
API
  • $(el).popConfirm(config)

config 参数如下：

参数 说明 类型 默认值
trigger 引发、触发,可选 click、hover、no,若设置为no时，说明已经在其它地方写入了事件，当前不需要再加事件 string click
title 标题 string -
content 内容 string -
width 宽度 string auto
height 高度 string auto
position 显示位置,可选 top-right、top、top-left、left、right、bottom-right、bottom、bottom-left string auto
closeable 是否显示关闭图标 boolean false
isArrow 是否显示箭头 boolean true
isMulti 是否显示多个框 boolean false
isShowIcon 是否显示warning图标 boolean false
isShowBtn 是否显示confirm按钮 boolean false
hide 点击页面其他地方是否隐藏popbox气泡框 boolean false
delay 鼠标滑动到popbox气泡框区域，多少秒后自动关闭，若设为0不关闭 number 300
okText 确认按钮文字 string 确定
cancelText 取消按钮文字 string 取消
onConfirm 点击确认的回调 function -
onClose 点击取消的回调 function -
基本

最简单的用法。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="popbox-basic">点击</button>

JavaScript 代码

$("#popbox-basic").popConfirm({
    title:'标题',
    content:'
content 内容
content 内容
',
    isShowBtn:true,
    onConfirm: function(){
        $.simpleAlert("您选择了确认");
    },
    onClose : function(){
        $.simpleAlert("您选择了取消");
    }
});
自定义按钮文字

设置属性 okText 和 cancelText 自定义按钮文字

HTML 代码

<button type="button" class="awsui-btn" id="custom-click">Click me</button>

JavaScript 代码

$("#custom-click").popConfirm({
    title:'标题',
    content:'
content 内容
content 内容
',
    isShowBtn:true,
    okText:'YES',
    cancelText:'NO',
    onConfirm: function(){
        $.simpleAlert("您选择了YES");
    }
});

HTML 代码

$("#icon-click").popConfirm({

content 内容

});

关闭图标

是否显示关闭图标，设置属性closeable:true。

HTML 代码

<button type="button" class="awsui-btn" id="close_icon">点击</button>

JavaScript 代码

$("#close_icon").popConfirm({
    title:'标题',
    content:'
content 内容
content 内容
',
    closeable:true,
    isShowIcon:true,
    isShowBtn:true,
    onConfirm: function(){
        $.simpleAlert("您选择了确认");
    }
});
