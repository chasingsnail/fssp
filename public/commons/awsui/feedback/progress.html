进度条

在操作需要较长时间才能完成时，为用户显示该操作的当前进度和状态。

  • 当一个操作会打断当前界面，或者需要在后台运行，且耗时可能超过2秒时。
  • 当需要显示一个操作完成的百分比时。
API
  • $.progress(config)

config 参数如下：

参数 说明 类型 默认值
elementId 元素id，必填 string -
type 类型，可选line 、 circle string line
percent 进度百分比 number 0
strokeWidth 进度条线的宽度，单位 px number 8
color 进度条颜色和字体库图标颜色 string -
icon isShowInfo为true时，状态图标请参考字体库图标 string -
width (type=circle) 圆形进度条画布宽度，单位 px number 100
isShowInfo 是否显示进度数值或状态图标 boolean true
进度条

标准的进度条。

  • 显示进度数值，设置parent属性
  • 显示状态图标，设置icon和color属性
  • 进度数值和状态图标都不显示，设置isShowInfo：false

</div>
<div class="awsui-code-demo" id="progress-demo-basic1" style="padding:0;">

</div>
<div class="awsui-code-demo" id="progress-demo-basic3" style="padding:0 0 14px;">
</div>

JavaScript 代码

$.progress({
    elementId:'progress-demo-basic',
    percent:30
});
$.progress({
    elementId:'progress-demo-basic1',
    percent:50,
    icon:"&#58927;",
    color:"#f78577"
});
$.progress({
    elementId:'progress-demo-basic2',
    percent:100,
    icon:"&#60017;",
    color:"#83d596"
});
$.progress({
    elementId:'progress-demo-basic3',
    percent:60,
    isShowInfo:false
});
