区域提示

区域提示向用户显示警告提示，经常用于以下情况：

  • 当某个页面需要向用户显示警示的信息时。
  • 非浮层的静态展现形式，始终展现，不会自动消失，用户可以点击关闭。
API
  • $(el).scopedNotifications(config)

config 参数如下：

参数 说明 类型 默认值
type 指定区域提示的类型，有四种选择ok、info、warning、error string info
content 区域提示内容 string -
description 区域提示的辅助性文字介绍 string -
fontSize 字体库图标大小 number 30
isShowIcon 是否显示type类型为ok、info、warning、error的字体库图标 boolean false
isShowClose 是否显示关闭按钮 boolean false
isShowTop 是否在顶部提示 boolean false
onClose 触发回调函数 function -
基本

最简单的用法，适用于简短的警告提示。

HTML 代码

<div class="awsui-code-demo" id="basic_demo"></div>

JavaScript 代码

$("#basic_demo").scopedNotifications({
    content:'区域提示内容'
});
信息提示类型

共有四种类型样式 ok、info、warning、error。

HTML 代码

<div class="awsui-code-demo" id="style_demo"></div>

JavaScript 代码

$("#style_demo").scopedNotifications({
    type:'ok',
    content:'ok 区域提示内容'
});
$("#style_demo").scopedNotifications({
    type:'info',
    content:'info 区域提示内容'
});
$("#style_demo").scopedNotifications({
    type:'warning',
    content:'warning 区域提示内容'
});
$("#style_demo").scopedNotifications({
    type:'error',
    content:'error 区域提示内容'
});
可关闭的区域提示

显示关闭按钮，点击可关闭区域提示，设置属性isShowClose：true。

HTML 代码

<div class="awsui-code-demo" id="close_demo"></div>

JavaScript 代码

$("#close_demo").scopedNotifications({
    content:'是否显示关闭按钮',
    description:'是否显示关闭按钮的辅助性文字介绍。',
    isShowClose:true,
    onClose : function(){
        console.log('I was closed.');
    }
});

顶部公告

用作顶部公告时，设置isShowTop 为 true。

HTML 代码

<div class="awsui-code-demo" id="top_demo"></div>

JavaScript 代码

$("#top_demo").scopedNotifications({
    type:'warning',
    content:'顶部的区域提示',
    description:'顶部的区域提示的辅助性文字介绍。',
    isShowTop:true
});
$("#top_demo").scopedNotifications({
    type:'warning',
    content:'顶部的区域提示',
    description:'顶部的区域提示的辅助性文字介绍。',
    isShowTop:true,
    isShowIcon:true,
    isShowClose:true
});
