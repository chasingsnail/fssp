侧边栏

一种页面展现方式，从屏幕侧面划出。使用简单，可以快速的制作出侧边栏展开收缩js特效。你可以在同一个页面实例化多个侧边栏实例，还可以设置侧边栏的响应式宽度，自定义侧边栏样式等。

API
  • $.openSidebar(config)

config 参数如下：

参数 说明 类型 默认值
url 侧边栏需要展示的URL地址 string -
title 侧边栏头部的标题 string -
closeText 侧边栏头部的关闭按钮的提示文字 string -
width 宽度，支持百分比和固定数字 string -
isMode 是否打开遮罩 boolean false
duration 侧边栏展开速度 string slow
iframeId 使用iframe容器时指定对应iframe的ID string -
containerId 使用div容器时指定对应div的ID string -
color 头部的颜色 string -
buttons 头部按钮 Array -
complete 在动画完成时执行的函数 function -
onClose 侧边栏收起时的回调函数 function -

 注意

iframe和div两种容器各对应了一个id，即iframeId和containerId，两种容器方式选择一种，两个同时设置默认使用iframeId。

基本用法

最简单的用法。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="basic_demo">侧边栏打开炎黄盈动官网</button>

JavaScript 代码

$("#basic_demo").on("click", function(){
	$.openSidebar({
		url : "http://www.actionsoft.com.cn",
		title : "炎黄盈动官方网站",
		closeText : "收起",
		width : "80%",
		isMode : false,
		duration : "slow",
		color:"#fff",
		iframeId : "iframeId",
		onClose : function(){
			console.log("I was closed.");
		}
	});
});
打开带有遮罩的侧边栏

打开带有遮罩的侧边栏，配置属性isMode : true，并展示页面内容。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="shade_demo">打开带有遮罩的侧边栏</button>

JavaScript 代码

$("#shade_demo").on("click", function(){
	$.openSidebar({
		url : "",
		title : "一个DIV容器的侧边栏",
		closeText : "收起",
		width : "40%",
		isMode : true,
		parent : parent,
		color : "#ffffff",
		duration : "slow",
		containerId : "sidebar_content",
		complete : function() {
			window.parent.document.getElementById("sidebar_content").innerHTML="
我是侧边栏内容
";
		}
	});
});
打开带有按钮的侧边栏

打开带有按钮的侧边栏，配置属性buttons，来完成按钮配置。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="button_demo">打开带有按钮的侧边栏</button>

JavaScript 代码

$("#button_demo").on("click", function(){
	$.openSidebar({
		url : "http://www.actionsoft.com.cn",
		title : "测试侧边栏",
		closeText : "收起",
		width : "60%",
		isMode : false,
		duration : "slow",
		iframeId : "frameId",
		topZone:"show",
		color : "#fff",
		buttons: [
			{
				id: "prev",
				text: "上一页",
				display: true,
				click: function () {
					alert(1)
				}
			},
			{
				id: "next",
				text: "下一页",
				display: true,
				click: function () {
					alert(2)
				}
			}
		],
		complete : function() {
			//侧边栏加载完毕的回调函数
		}
	});
});
