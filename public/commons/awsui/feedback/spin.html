加载中

页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

API
  • $(el).loading(config)

config 参数如下：

参数 说明 类型 默认值
size 组件大小，可选值为 small default large string default
description 当作为包裹元素时，可以自定义描述文案 string -
color 描述文案字体颜色 string -
delay 延迟显示加载效果的时间 number(毫秒) -
hidden 是否隐藏loading boolean false
基本用法

一个简单的 loading 状态。

HTML 代码

<div class="awsui-code-demo" id="loading-demo-basic"></div>

JavaScript 代码

$("#loading-demo-basic").loading();
各种大小

小的用于文本加载，默认用于卡片容器级加载，大的用于页面级加载。

HTML 代码

<div class="awsui-code-demo" id="loading-demo-size"></div>

JavaScript 代码

$("#loading-demo-size").loading({
    size:'small'
});
$("#loading-demo-size").loading({
    size:'default'
});
$("#loading-demo-size").loading({
    size:'large'
});
自定义提示语

自定义描述loading文案。

HTML 代码

<div class="awsui-code-demo awsui-align-center" id="loading-demo-custom"></div>

JavaScript 代码

$("#loading-demo-custom").loading({
    description:"loading",
    color:'#3983de'
});
延迟

延迟显示 loading 效果。

HTML 代码

<div class="awsui-code-demo awsui-align-center" id="loading-demo-delay"></div>

JavaScript 代码

$("#loading-demo-delay").loading({
    delay:2000
});
窗口加载

窗口loading。

HTML 代码

<div class="awsui-code-demo awsui-align-center" id="loading-demo-window"></div>

JavaScript 代码

$("#loading-demo-window").loading();
弹出框加载

弹出框loading，一般适用于查询信息。

HTML 代码

<div class="awsui-code-demo" id="loading-demo-search">
	<button type="button" class="awsui-btn type" value="loading">loading</button>
</div>

JavaScript 代码

$("#loading-demo-search").find("button").on("click", function(){
	$.simpleAlert("正在查询......",'loading');
	setTimeout('$.simpleAlert("close")',5000);
});
loading消失

设置属性hidden为true，则不显示 loading 状态。

HTML 代码

<div class="awsui-code-demo awsui-align-center" id="loading-demo-hide"></div>

JavaScript 代码

$("#loading-demo-hide").loading({
    hidden:true
});
