文字提示

鼠标停留在元素上面时触发文字提示框，通过给标签加awsui-qtip=""属性来实现。

API
  • $(el).tooltip(config)

config 参数如下：

参数 说明
text 提示框文字内容
position 提示框显示位置
bordercolor 边框颜色
bgcolor 背景颜色
color 文字颜色
close 是否出现关闭图标
delay 自动延迟时间关闭
提示框位置

HTML 代码

<button type="button" id="bt1" awsui-qtip="text:'上面提示框',
 position:'top'" class="awsui-btn awsui-btn-normal">top</button>
<button type="button" id="bt2" awsui-qtip="text:'右面提示框',
 position:'right'" class="awsui-btn awsui-btn-normal" >right</button>
<button type="button" id="bt3" awsui-qtip="text:'下面提示框',
 position:'bottom'" class="awsui-btn awsui-btn-normal">bottom</button>
<button type="button" id="bt4" awsui-qtip="text:'左面提示框',
 position:'left'" class="awsui-btn awsui-btn-normal">left</button>
