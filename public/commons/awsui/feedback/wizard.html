向导对话框

平台Wizard向导对话框组件(卡片布局)。

  • 用于引导教程、多页注册、上传等场景。
API
  • $(el).wizard(config)

config 参数如下：

参数 说明 类型 默认值
width 设置宽度 number -
height 设置高度 number -
color 设置当前步骤和已完成步骤的颜色 string #3983de
currentStep 当前步骤 number 0
nextLabel 下一步按钮的标签名 string [下一步]
nextCls 下一步按钮的样式class名 string button
prevLabel 上一步按钮的标签名 string [上一步]
prevCls 上一步按钮的样式class名 string button
finishLabel 完成按钮的标签名 string button
finishCls 完成按钮的样式class名 string button
items [ {title:"标题1",des:"内容1"}, {title:"标题2",des:"内容2"} ] Array []
onNext 点击下一步执行的事件,参数为组件配置参数和动态配置参数 function -
onPrev 点击上一步执行的事件,参数为组件配置参数和动态配置参数 function -
onFinish 点击完成按钮执行的事件,参数为组件配置参数和动态配置参数 function -
btnHandle 显示和隐藏按钮方法 function -
引入文件

务必确保路径正确

<script src="../commons/js/jquery/scripts/ui/aws.wizard.js"></script>

最标准的用法。

 注意

1、要给dialog()配置参数，必须先调用$(el).dialog()方法，再调用$(el).wizard()方法

2、若不给dialog()配置参数，不要调用$(el).dialog()方法，直接调用$(el).wizard()方法

JavaScript 代码

<script type="text/javascript">
    var 下一步 = "下一步";
    var 上一步 = "上一步";
    var 完成 = "完成";
    $(document).ready(function() {
        var options = {
            color:"#3983de",
            currentStep:2,
            items:[
                {title:"第一步",des:"这是第一步加载的内容"},
                {title:"第二步",des:""},
                {title:"第三步",des:"这是第三步加载"},
                {title:"第四步",des:""}
            ],
            onNext:function(options){
                $.simpleAlert("进行到第" + (options.currentStep + 2) + "步");
                return true;
            },
            onPrev:function(options){
                $.simpleAlert("进行到第" + (options.currentStep) + "步");
                return true;
            },
            onFinish:function(options){
                $.simpleAlert("进行到第" +(options.currentStep+1)+"步，3秒钟后关闭", "info",3000, {callback:function(){
                        $("#wizard-dig").dialog("close");}
                });
                return true;
            }
        };

        $("#show").on("click", function(){
            $("#awsui-wizard").wizard(options);
        });
    });
</script>
