手风琴

手风琴允许用户切换一部分内容的显示。

  • 对复杂区域进行分组和隐藏，保持页面的整洁。
API
  • $(el).accordions(config)

config 参数如下：

参数 说明 类型 默认值
background 背景颜色 string #f8f8f8
color 字体库图标颜色 string #666
fontSize 字体库图标大小 number 16
icon 字体库图标 string &#xe716;
currentIcon 当前选中字体库图标 string &#xe717;
list accordion列表 Array []
expand 展开形式, all/first/collapse 全部展开、展开首个、全部收起 string collapse
speed 收起/展开速度 number 0 单位是ms

方法如下：

方法名 说明 类型
expand(index) 展开指定区域 function
collapse(index) 收起指定区域 function
expandAll() 全部展开 function
collapseAll() 全部收起 function
基础

简单的手风琴。

HTML 代码

<ul class="awsui-collapse" id="accordion1"></ul>

JavaScript 代码

$(document).ready(function() {
    $("#accordion1").accordions({
        list:[
            {title:"百度",content:"accordion内容接受任何布局或组件"},
            {title:"腾讯",content:"accordion内容接受任何布局或组件"},
            {title:"阿里",content:"accordion内容接受任何布局或组件"}
        ]
    });
});
改变风格

通过设置属性background、color、icon、fontSize来改变手风琴风格。

HTML 代码

<ul class="awsui-collapse" id="accordion2"></ul>

JavaScript 代码

$(document).ready(function() {
    $("#accordion2").accordions({
        background:"#4db763",
        color:"#fff",
        icon:"&#58888;",
        currentIcon:"&#58936;",
        fontSize:20,
        expand:"all",
        speed:500,
        list:[
            {title:"百度",content:"accordion内容接受任何布局或组件"},
            {title:"腾讯",content:"accordion内容接受任何布局或组件"},
            {title:"阿里",content:"accordion内容接受任何布局或组件"}
        ]
    });
});
全部展开和全部收起

全部展开和全部收起手风琴。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="expandAll">全部展开</button>
<button type="button" class="awsui-btn awsui-btn-blue" id="collapseAll">全部收起</button>
<ul class="awsui-collapse" id="accordion3" style="margin-top:10px;">
</ul>

JavaScript 代码

$(document).ready(function() {
    var accordion =  $("#accordion3").accordions({
        list:[
            {title:"百度",content:"accordion内容接受任何布局或组件"},
            {title:"腾讯",content:"accordion内容接受任何布局或组件"},
            {title:"阿里",content:"accordion内容接受任何布局或组件"}
        ],
        speed:0
    });
    $("#expandAll").on('click',function () {
        accordion.expandAll();
    });
    $("#collapseAll").on('click',function () {
        accordion.collapseAll();
    });
});
指定区域展开和收起

指定区域展开和收起。

HTML 代码

<button type="button" class="awsui-btn awsui-btn-blue" id="expand">指定展开</button>
<button type="button" class="awsui-btn awsui-btn-blue" id="collapse">指定收起</button>
<ul class="awsui-collapse" id="accordion4" style="margin-top:10px;">
</ul>

JavaScript 代码

$(document).ready(function() {
    var accordion4 =  $("#accordion4").accordions({
        list:[
            {title:"百度",content:"accordion内容接受任何布局或组件"},
            {title:"腾讯",content:"accordion内容接受任何布局或组件"},
            {title:"阿里",content:"accordion内容接受任何布局或组件"}
        ],
        speed:500
    });
    $("#expand").on('click',function () {
        var index = $("#accordion4").find("li").eq(1).index();
        accordion4.expand(index);
    });
    $("#collapse").on('click',function () {
        var index = $("#accordion4").find("li").eq(1).index();
        accordion4.collapse(index);
    });
});
