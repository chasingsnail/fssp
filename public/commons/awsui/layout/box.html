盒子

日常生活中所见的盒子也就是能装东西的一种箱子，把HTML页面中的元素看作是一个矩形的盒子，也就是一个盛装内容的容器，称它为盒子模型。

  • 盒子区域，可以装载图片、文字等页面元素。
API
  • $(el).box(config)

config 参数如下：

参数 说明 类型 默认值
id box的唯一标识，id string -
width box的宽度 number 默认适应内容
height box的高度 number 默认适应内容
background box区域选中时背景颜色 string #f9f9f9
fontSize 字体库图标大小 number 20
color 字体库图标颜色 string #0ca72d
icon 字体库图标（默认的是选中图标，若想改图标请修改图标对应的Unicode字符） string &#60017;
isClickSelected 点击当前区域是否选中，并显示选中图标 boolean false
isSelected 默认当前区域选中 boolean false
isMultipleSelected 是否允许多个box被选中 boolean false
disabled box区域为只读效果 boolean false
noBorder box区域无边框 boolean false
onClick box点击事件 function -

 注意，禁用和只读效果，提供了快捷方式，如：

1、禁用效果，调用$(el).box("disabled",true)方法

2、只读效果，调用$(el).box("readonly",true)方法

点击Box

点击Box - 一个含有基本元素，支持鼠标hover和点击事件的box,可组合成如下场景：

  • box区域宽度可自定义，设置属性width的值
  • box区域选中时改变背景颜色，设置属性background的值
  • 设置属性color,即可改变字体图标的颜色
  • 设置属性icon,即可改变时字体图标
  • 设置属性fontSize,即可改变时字体图标大小
  • 设置属性disabled为true,即该区域为只读效果
