页面布局

进行页面级整体合理布局。

API
  • $(el).layout(config)

layout()方法，设置传入方法中的参数，即可进行页面布局，config 参数如下：

参数 说明 类型 默认值
left config属性之一，其结构为left:{target:""} object -
right config属性之一，其结构为right:{target:""} object -
open config属性之一，其结构为open:true/false boolean false
上中下布局

最基本的『上-中-下』布局。

一般主导航放置于页面的顶端，从左自右依次为：logo、一级导航项、辅助菜单（用户、设置、通知等）。上下级的结构符合用户上下浏览的习惯，也是较为经典的网站导航模式。页面上下切分的方式提高了主工作区域的信息展示效率，但在纵向空间上会有一些牺牲。此外，由于导航栏水平空间的限制，不适合那些一级导航项很多的信息结构。

JavaScript 代码

$(document).ready(function(){
    $(".awsui-demo2 li").on("click",function(){
        $(".awsui-demo2 li").each(function(){
            $(this).removeClass("awsui-header-this");
        });
        $(this).addClass("awsui-header-this");
    });
    $("#awsui-demo2-left li").children("ul").children("li").on("click",function(){
        $("#awsui-demo2-left li").children("ul").children("li").each(function(){
            $(this).removeClass("awsui-left-bg-this");
            $(this).children("span").eq(0).removeClass("awsui-left-this");
            $(this).children("span").eq(1).removeClass("awsui-left-nav-this");
        });
        $(this).addClass("awsui-left-bg-this");
        $(this).children("span").eq(0).addClass("awsui-left-this");
        $(this).children("span").eq(1).addClass("awsui-left-nav-this");
	});
	$("#awsui-demo2-left li").children("div").on("click",function(){
		$(this).next().slideToggle(500).parent($(this)).siblings().children('ul');
	});
});
左右布局

侧边导航在页面布局上采用的是左右的结构，一般主导航放置于页面的左侧固定位置，辅助菜单放置于工作区顶部。内容根据浏览器终端进行自适应，能提高横向空间的使用率，但是整个页面排版不稳定。 侧边导航的模式层级扩展性强，一、二、三级导航项目可以更为顺畅且具关联性的被展示，同时侧边导航可以固定，使得用户在操作和浏览中可以快速的定位和切换当前位置，有很高的操作效率。

var options = {
	left:{
		target:"awsui-demo3-left"
	}
};
$("#awsui-demo3").layout(options);
$("#" + options.left.target).children("li").children("div").on("click",function(){
	$("#" + options.left.target).children("li").children("div").each(function(){
		$(this).children("span").removeClass("awsui-left-this");
	});
	$(this).children("span").addClass("awsui-left-this");
});
