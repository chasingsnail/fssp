面板

面板通常用于提供补充信息或表单与主画布相关的输入，面板应占用其父容器高度的100％。在大多数情况下，这将是您的应用程序的视口或主要阶段。

  • 画布内容超出了视口的边界，同时支持平移和缩放。

标题

标题停靠在其面板的顶部。当面板主体awsui-panel-body给出固定高度时，面板主体的内容溢出时，主体将提供滚动，同时标题保持可见的位置。

标题默认为左对齐文本，通过使用awsui-ellipsis类，当标题宽度对于面板宽度变得太长时，标题被截断，出现省略号……

面板可见性

面板是否可见，只需将类awsui-show(显示)或awsui-hide(隐藏)添加到awsui-panel元素即可。

面板可见性 
页面内容

HTML 代码

<div class="awsui-public-border awsui-public-radius" style="padding: 10px;">
    <button type="button" class="awsui-btn awsui-btn-blue" id="isShowBtn">可见性</button>
    <div class="awsui-public-border awsui-public-radius awsui-margin-top-10 awsui-background" style="display: flex; height: 80px;">
        <div id="panelIsShow" class="awsui-panel awsui-size-medium awsui-panel-docked awsui-panel-docked-left" data-hide="false">
            <div class="awsui-panel-header awsui-align-center">
                <span class="awsui-panel-header-title">面板可见性</span>
                <span class="awsui-iconfont awsui-public-box-close">&#58931;</span>
            </div>
            <div class="awsui-panel-body">
            </div>
        </div>
        <div class="awsui-around-medium">
            页面内容
        </div>
    </div>
</div>

JavaScript 代码

$(document).ready(function() {
    $("#isShowBtn,.awsui-public-box-close").on("click", function () {
        var panel = $("#panelIsShow");
        if (panel.data('hide')) {
            panel.data('hide',false);
            panel.css("display","block");
        } else {
            panel.data('hide',true);
            panel.css("display","none");
        }
    });
});
