静态表格

风格多样的静态表格,展示行列数据；可用于以下场景：

  • 当有大量结构化的数据需要展现时。
  • 当需要对数据进行排序、搜索、分页、自定义操作等复杂行为时。
CSS
class名称 说明
awsui-table-left 内容左对齐
awsui-table-center 内容居中
awsui-table-right 内容右对齐
awsui-table 标准样式，带边框
awsui-table-condensed 紧致边框
awsui-table-thin 仅有横线的边框
awsui-table-ordinate 仅有纵线的边框
awsui-table-hover 鼠标悬停
awsui-table-striped 斑马条
